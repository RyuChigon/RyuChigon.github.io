## 풀이 방식

Left -> Right로 증가하는 수와, Right -> Left로 증가하는 수를 Index별로 구한 후,

합을 구한다.

## 코드

```python
from bisect import bisect_left
n = int(input())
nums = list(map(int, input().split()))
cnt = [0] * n
temp = [float('inf')] * n

max_idx = -1
for i in range(n):
    insert_idx = bisect_left(temp, nums[i])
    temp[insert_idx] = nums[i]
    max_idx = max(max_idx, insert_idx)
    cnt[i] += max_idx

max_idx = -1
temp = [float('inf')] * n
for i in range(n - 1, -1, -1):
    insert_idx = bisect_left(temp, nums[i])
    temp[insert_idx] = nums[i]
    max_idx = max(max_idx, insert_idx)
    cnt[i] += max_idx

print(max(cnt) + 1)
```
