## 풀이 방식

1. 구간 합을 구한다.
2. 나머지가 같은 두 구간합은 나머지가 0인 것을 이용한다.
3. dict 구조를 이용하여 나머지가 0일 때 카운트를 한다.

## 코드

```python
from collections import defaultdict

n, m = map(int, input().split())
nums = list(map(int, input().split()))
reminder = defaultdict(int)
reminder[0] = 1
pre_sum = 0
for i in range(n):
    pre_sum = (pre_sum + nums[i]) % m
    nums[i] = pre_sum

cnt = 0
for num in nums:
    cnt += reminder[num]
    reminder[num] += 1
print(cnt)
```

## 후기

구간 합 유형은 난이도 있는 문제에 속한다. 구간 합 개념을 알고 있으면 풀 수 있는 문제가 많이 생기는 것 같다.
